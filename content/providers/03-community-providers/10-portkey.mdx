# Portkey Provider

[Portkey](https://portkey.ai/) is an AI Gateway that allows developers to connect to many AI models with a single AI SDK provider.
Portkey also brings in observability, caching, and other features that are useful for building production-grade AI applications.

## Setup

The Portkey provider is available in the `@portkey-ai/vercel-provider` module. You can install it with

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    <Snippet text="pnpm add @portkey-ai/vercel-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @portkey-ai/vercel-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @portkey-ai/vercel-provider" dark />
  </Tab>
</Tabs>

## Provider Instance

You can create the Portkey provider instance using `portkey`:

```ts
import { portkey } from '@portkey-ai/vercel-provider';
```

If you need a customized setup, you can import `createPortkey` from `@portkey-ai/vercel-provider`:

```ts
import { createPortkey } from '@ai-sdk/portkey';

const portkey = createPortkey({
  apiKey: process.ENV.PORTKEY_API_KEY,
  config: process.PORTKEY_CONFIG_ID,
});
```

## Language Models

The **[Portkey provider](https://sdk.vercel.ai/providers/community-providers/portkey)** for the AI SDK
contains language model support for the Portkey chat and completion APIs.

### Example

```ts
import { portkey } from '@portkey-ai/vercel-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: portkey('gpt-4-turbo'),
  prompt: 'What is a portkey?',
});
```

For more examples, please refer to the [Portkey documentation](https://docs.portkey.ai/docs/integrations/libraries/vercel).
